
This is a TypeScript library based on npm and vite that provides a treemap visualization rendering system.
The base dependencies are webgl-operate (and thus WebGL).
Next to the library itself, it contains a set of examples for testing and showcasing.

Follow [REQUIREMENTS.md](./REQUIREMENTS.md) to find the functional requirements for this project.
Follow [NONFUNCTIONAL_REQUIREMENTS.md](./NONFUNCTIONAL_REQUIREMENTS.md) to find the architectural and non-functional requirements for this project.
