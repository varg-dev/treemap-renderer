# Build Better Treemaps with @varg-dev/treemap-renderer

`@varg-dev/treemap-renderer` is a WebGL-based library for high-performance, research-driven treemap rendering in the browser.

## Why use it?

- Fast rendering pipeline built on top of `webgl-operate`.
- Multiple layout algorithms: Strip, Snake, Hilbert, Moore, and Code City.
- Flexible data ingestion via direct config and CSV-based frontends.
- Rich visual encodings for color, labels, emphasis, and geometry.
- Example gallery included to get from dataset to visualization quickly.

## Typical use cases

- Interactive hierarchical data exploration in dashboards.
- Visual analytics prototypes and research tooling.
- Teaching and demonstration of treemap layout variants.
